# «Будущее в моих руках» — Визуальная новелла на Ren'Py

## О проекте

**Жанр:** Визуальная новелла / интерактивная история  
**Движок:** Ren'Py 8.x  
**Тема:** «Как достичь успеха, если ты студент»  
**Контент:** Сгенерирован с помощью нейросетей (Pillow + алгоритмические методы)

## Сюжет

**Главный герой:** Алекс — студент-первокурсник IT-специальности.

### 12 сцен:
1. **Общежитие** — Первое утро. Знакомство с обстановкой.
2. **Коридор** — Встреча с Машей, подругой.
3. **Аудитория** — Первая лекция профессора Коваля.
4. **Библиотека** — **Развилка №1**: учиться или гулять?
5. **Кафе** — Знакомство с Димой и Катей.
6. **Ночной город** — **Развилка №2**: выбор стратегии (учёба/хакатон/стажировка).
7. **Кабинет профессора** — Разговор с Ковалём.
8. **Крыша** — Разговор с Катей о будущем.
9. **Парк** — Маша в беде. **Развилка №3**: помочь или нет?
10. **Зал экзамена** — Кульминация. Экзамен.
11. **Будущий город** — Видение будущего.
12. **Выпускной** — Финал. Одна из 4 концовок.

### 4 концовки:
- 🏆 **«Восхождение»** — Максимальный успех (учился + помог Маше + поговорил с профессором)
- 📚 **«Твёрдый шаг»** — Средний результат (честный путь)
- 🔄 **«Второй шанс»** — Провал и пересдача (возможность роста)
- 🌟 **«Свой путь»** — Нестандартный выбор (академический отпуск)

## Механика

- **Переменные выбора:** `has_studied`, `befriended_masha`, `helped_prof`, `impressed_dima`, `romance_katya`
- **Счётчик:** `final_grade` (0–14 баллов) определяет концовку
- **Ветвление:** 3 основных развилки + мини-выборы в сценах

## Установка и запуск

1. Скачайте Ren'Py с [renpy.org](https://www.renpy.org/latest.html)
2. Поместите папку `renpy_project` в директорию `Projects` Ren'Py
3. Выберите проект в лаунчере и нажмите **«Запустить»**

### Добавление музыки (опционально):
Поместите `.ogg` файлы в `game/audio/music/` и раскомментируйте строки `# play music` в `script.rpy`.
